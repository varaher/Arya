### 2. Marathi typed text returns Hindi

```
Current:  Devanagari detection → always "hi"
          mr-IN voice → correctly "mr"
          But typed Marathi → wrongly "hi"

Problem:  83 million Marathi speakers
          typing in ARYA get Hindi responses

Fix (add after Devanagari detection):
```

```typescript
function disambiguateDevanagari(text: string): string {
  const marathiMarkers = [
    'आहे', 'नाही', 'आणि', 'करा', 'होते',
    'तुम्ही', 'आम्ही', 'काय', 'कसे', 'मला'
  ];
  const hindiMarkers = [
    'है', 'नहीं', 'और', 'करो', 'था',
    'तुम', 'हम', 'क्या', 'कैसे', 'मुझे'
  ];

  const words = text.split(/\s+/);
  const mrScore = words.filter(w => marathiMarkers.includes(w)).length;
  const hiScore = words.filter(w => hindiMarkers.includes(w)).length;

  if (mrScore > hiScore) return 'mr';
  if (hiScore > mrScore) return 'hi';
  return 'hi'; // default if tied
}
```

Move this from lower priority to this sprint. 83 million speakers is not lower priority.
