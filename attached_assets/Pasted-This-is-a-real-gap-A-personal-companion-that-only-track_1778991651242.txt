This is a real gap. A personal companion that only tracks reading habits as goals is not a personal companion — it's a habit tracker.

Let me map out exactly what needs to change.

---

## What's Wrong Right Now

The goal detection in `chat-engine.ts` uses pattern matching that's biased toward wellness and habit-style goals. Anything that doesn't sound like a habit — "message Priya about the meeting", "follow up with the investor", "buy groceries before Thursday" — either gets missed entirely or gets forced into a habit template that doesn't fit.

```
Current goal model (implicit):
  - Habit-style goals with streaks
  - Daily recurrence assumed
  - Progress tracked by check-in frequency
  - Examples: walk 30 mins, read 20 mins, meditate

What it should be:
  - ANY intention the user expresses
  - One-time tasks
  - Reminders with specific times
  - Long-term goals with milestones
  - Soft intentions ("I want to call mom more")
  - Hard deadlines ("submit proposal by Friday")
```

---

## The Four Goal Types ARYA Should Recognise

```
Type 1 — TASK (one-time, completable)
  "Message Priya about the project"
  "Buy dad a birthday gift"
  "Follow up with the investor this week"
  → No streak. Just done or not done.
  → Can have a deadline.

Type 2 — HABIT (recurring, streak-trackable)
  "Walk 30 minutes every morning"
  "Read before bed"
  "Drink 2 litres of water daily"
  → Has recurrence. Streak makes sense.
  → Current model handles this well.

Type 3 — REMINDER (time-specific)
  "Remind me to call Rajan at 3pm"
  "Message the team before the meeting tomorrow"
  "Pay the electricity bill before the 15th"
  → Has a specific time or date.
  → Should fire a notification.
  → Google Calendar connection relevant here.

Type 4 — INTENTION (soft, no deadline)
  "I want to spend more time with family"
  "I should be more patient with myself"
  "I want to learn something new this month"
  → No hard deadline or recurrence.
  → ARYA checks in on these periodically.
  → Tracks sentiment, not completion.
```

## Google Calendar Integration — How It Connects

You already have Calendar OAuth. Wire goal creation to it.

```typescript
// server/arya/goal-calendar-sync.ts

async function syncGoalToCalendar(
  goal: Goal,
  userId: string
): Promise<string | null> {

  // Only sync goals with time component
  if (!goal.due_date && !goal.reminder_time) return null;

  // Only if user has calendar connected
  const calendarConnected = await isCalendarConnected(userId);
  if (!calendarConnected) return null;

  const eventTime = goal.reminder_time || goal.due_date;

  const calendarEvent = {
    summary: `ARYA: ${goal.title}`,
    description: goal.context_note || '',
    start: {
      dateTime: eventTime,
      timeZone: await getUserTimezone(userId)
    },
    end: {
      dateTime: addMinutes(eventTime, 30), // 30 min default
      timeZone: await getUserTimezone(userId)
    },
    reminders: {
      useDefault: false,
      overrides: [
        { method: 'popup', minutes: 30 },
        { method: 'popup', minutes: 5 }
      ]
    }
  };

  const event = await createCalendarEvent(userId, calendarEvent);
  return event.id; // stored in goal.calendar_event_id
}
```

## The Reminder Firing — Using Existing Scheduler

You already have `reminder-scheduler.ts`. Add a goal reminder check.

```typescript
// Add to reminder-scheduler.ts

async function checkGoalReminders() {
  const now = new Date();
  const upcoming = await db.query(`
    SELECT g.*, u.id as user_id
    FROM arya_goals g
    JOIN arya_users u ON g.user_id = u.id
    WHERE g.goal_type IN ('task', 'reminder')
      AND g.reminder_time IS NOT NULL
      AND g.reminder_time <= NOW() + INTERVAL '5 minutes'
      AND g.reminder_time >= NOW() - INTERVAL '5 minutes'
      AND g.reminder_fired = false
      AND g.is_completed = false
  `);

  for (const goal of upcoming) {
    await sendNotification(goal.user_id, {
      type: 'goal_reminder',
      title: `ARYA reminder`,
      body: goal.title,
      data: { goal_id: goal.id }
    });

    await db.query(`
      UPDATE arya_goals
      SET reminder_fired = true
      WHERE id = $1
    `, [goal.id]);
  }
}

// Run every 5 minutes alongside existing checks
```
